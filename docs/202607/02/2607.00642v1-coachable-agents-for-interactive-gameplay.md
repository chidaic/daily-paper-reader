## 摘要
在创建先进AI和机器人系统方面，强化学习已被证明是一种有价值的工具，其贡献涵盖了从游戏到机器人再到基础模型的各个领域。通过试错，这些AI系统通常学习一种近乎最优的行为来解决任务。然而，在许多用例中，人们希望在一定程度上实时控制任务解决的方式。我们将这些对核心任务的修改称为风格。我们将通用价值函数逼近器（UVFA）与精心选择的训练场景、学习算法和数据增强相结合，创建了一个教练框架，使代理能够在复杂领域中展现风格。我们展示了该框架在AAA视频游戏《地平线：西之绝境》和《Gran Turismo》以及一个开源的人形测试域中的应用。尽管这些领域性质不同——赛车、风格化游戏战斗和人形行走——每个代理都表现出对风格请求的强一致性，同时仍能满足其主要任务。重要的是，本文概述的技术允许最终用户在运行时选择最终行为，从而灵活控制最终执行的性能。

## Abstract
Reinforcement learning has proven to be a valuable tool in the creation of advanced AI and robotic systems, contributing to everything from game playing to robotics to foundation models. Through trial-and-error, these AI systems typically learn one, near-optimal behavior to solve their tasks. However, there are many use cases in which one would like to assert some level of control, preferably in real time, over how the task is solved. We refer to these modifications of a core task as styles. We combine universal value function approximators (UVFAs) with carefully selected training scenarios, learning algorithms, and data augmentation to create a framework for coaching agents that exhibit styles in complex domains. We demonstrate the framework's application in the AAA video games Horizon Forbidden West and Gran Turismo, and in an open-source humanoid test domain. Despite the different nature of the domains -- car racing, stylized game combat, and humanoid walking -- each agent shows strong coherence to the style requests while still satisfying the main task in its domain. Importantly, the techniques outlined in this paper allow an end user to choose the final behavior at run time, giving them flexible control over the final executed performance.

## 2. 论文提出的方法论

- **核心思想**：基于**通用值函数逼近器（UVFA）** 进行扩展，使其条件于**风格向量 θ**，而非传统的单一“目标状态”。智能体的策略 π(a|s, θ) 和价值函数 Q(s, a, θ) 均以状态和风格向量为条件，从而在单一网络中编码多种行为风格。
- **关键技术细节**：
  - **风格化奖励分解**：总奖励 r(s,a;θ) = r_task(s,a) + r_style(s,a,θ)。r_task固定不变，负责核心任务目标；r_style根据多维风格向量θ组合多个风格奖励函数（线性权重、阈值、目标设定点等）。
  - **训练方法**：
    - 使用**Soft Actor-Critic (SAC)** 及其变体（如QR-SAC、Categorical Regularized Actor-Critic, Cat-RAC）作为基础算法。
    - 引入**场景训练（Scenario Training）**：在训练中生成多种多样化的初始状态和场景，提升泛化能力。
    - **数据增强**：采用**任务冒充（Task Impersonation）**：对每一批经验，用多个随机采样的风格参数重新计算奖励，扩充训练信号。
    - **分层采样（Stratified Sampling）**：利用**多表回放缓冲（Multi-tables）** 或**事件表（Event Tables）**，按风格配置分组存储轨迹，并分层采样确保各场景和风格均衡学习。
  - **算法流程**：
    1. 在每个episode开始时，从分布 p(θ) 中采样一个风格向量 θ（例如，多个风格权重独立采样，或者以一个概率选择单一风格）。
    2. 智能体与环境交互，每一步获得 r_task 和 r_style 之和作为即时奖励。
    3. 将经验（s, a, r, s', θ）存储到回放缓冲（含事件表）。
    4. 从缓冲中采样批次，期间使用任务冒充来覆盖更多 θ 值。
    5. 更新策略和评论家网络，使其学会在给定 θ 下最大化期望回报。

- **核心公式**（用文字说明）：
  - 风格参数 θ 包含多个维度，每个维度对应一种行为偏好（如武器类型、元素属性等）。训练时，每个 θ 分量在一个连续区间内随机采样，或采用混合分布（如一个热采样、独立采样等）。
  - 评论家学习风格条件值函数 Q(s, a, θ)；策略输出动作时也条件于 θ。这使得同一模型在测试时只需改变 θ 即可切换行为。

## 8. 不足与局限

- **未与外部现有方法直接对比**：实验均为框架内消融，未与多任务RL、多目标RL、或基于条件策略的其他风格方法（如de Woillemont et al.的工作）进行公平比较，难以量化本方法的相对优势。
