## 摘要
我们研究当参与者将信念形成委托给预测性机器学习（ML）时的战略互动。在一个静态贝叶斯博弈中，每个参与者的ML代理预测一个与收益相关的结果变量，作为参与者类型的函数。ML代理的训练样本是内生的：它来自参与者ML引导行为所产生的结果分布。在交叉验证均衡（CVE）中，每个参与者的ML代理根据其实现的训练样本选择预测模型，以最小化期望的样本外平方误差，并且每个参与者对其ML代理所选模型产生的信念做出最优反应。我们分析了CVE并将其与其他均衡概念联系起来。我们将CVE应用于陪审团投票、投机性投注以及具有线性二次收益的博弈。例如，在团队努力博弈中，内生模型选择可能引发多重均衡。

## Abstract
We study strategic interaction when players delegate belief formation to predictive machine learning (ML). In a static Bayesian game, each player's ML agent predicts a payoff-relevant outcome variable as a function of the player's type. The ML agent's training sample is endogenous: it is drawn from the outcome distribution generated by players' ML-guided behavior. In Cross-Validation Equilibrium (CVE), each player's ML agent selects a predictive model to minimize expected out-of-sample squared error, given its realized training sample, and each player best-replies to the belief generated by the model her ML agent selected. We analyze CVE and relate it to other equilibrium concepts. We apply CVE to jury voting, speculative betting, and games with linear-quadratic payoffs. E.g., in a team-effort game, endogenous model selection can give rise to multiple equilibria.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：随着机器学习（ML）在信用评估、司法风险评估等经济决策中的广泛应用，人类决策者开始依赖ML代理形成的预测信念。然而，在博弈情境下，每个玩家的ML代理使用的训练样本并非外生给定，而是由所有玩家对自身ML信念的响应行为共同内生生成的。这种双向反馈（ML预测影响行为，行为又反过来影响ML的训练数据）使得传统博弈均衡分析面临挑战。
- **核心问题**：当每个玩家将信念形成委托给一个预测性ML代理（该代理基于从均衡分布中抽样的内生训练样本，选择最小化样本外平方预测误差的模型）时，均衡策略会如何？如何定义并求解这种新型均衡？
- **整体含义**：本文提出“交叉验证均衡”（Cross-Validation Equilibrium, CVE），为理解算法辅助决策下的战略互动提供了新框架，揭示了内生模型选择可能导致多重均衡、偏离纳什均衡等丰富现象。

## 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程（用文字说明）

- **核心思想**：
  - 考虑一个静态贝叶斯博弈，每个玩家 i 的收益依赖于一个标量充分统计量 xi（例如对手的行动、需求参数等）。玩家 i 委托一个ML代理，该代理基于训练样本（包含真实预期值加独立同分布零均值噪声）选择划分其类型空间 Ti 的最优分区模型 Π_i，以最小化期望样本外平方预测误差。玩家 i 再根据所选分区产生的信念来做出最优反应。
  - 由于训练样本是随机的，选出的分区和玩家的行为也是随机的，从而玩家采用混合策略。CVE要求：对于每个噪声实现 ε_i，ML代理的分区选择是样本外最优的，且玩家的行为是对该分区信念的最优反应；玩家的混合策略由噪声分布和分区选择概率整合而成。

- **关键技术细节**：
  - **训练样本**：对于每个类型 t_i，训练样本观测值为 y_{t_i} = \bar{x}_i^σ(t_i) + ε_{t_i}，其中 \bar{x}_i^σ(t_i) 是给定对手策略 σ_{-i} 下类型 t_i 的条件期望充分统计量，ε_{t_i} 独立同分布、对称、均值为零。
  - **模型选择**：ML代理从可行分区集合 P_i 中选择一个分区 Π_i，使得期望样本外平方预测误差最小。误差定义基于验证样本噪声 η_{t_i}（独立同分布零均值），评估时对验证噪声取期望。
  - **玩家最优反应**：给定所选分区 Π_i 和训练样本 ε_i，玩家 i 更新信念（使用分区内样本均值作为预测），并选择最大化期望效用的行动。由于 ui 关于 xi 线性，仅需预测 xi 的均值。
  - **均衡定义**：一个策略剖面 σ 是CVE，如果对于每个玩家 i 和每个训练样本实现 ε_i，存在一个混合策略 β_{ε_i} 支持在兼容对 (Π_i, g_i) 上，使得 σ_i(a_i | t_i) = ∑_{ε_i} υ(ε_i) ∑_{(Π_i,g_i): g_i(t_i)=a_i} β_{ε_i}(Π_i, g_i)。

- **算法流程**（文字描述）：
  1. 给定初始对手策略 σ_{-i}，计算每个类型 t_i 的 \bar{x}_i^σ(t_i)。
  2. 对于每个可能的训练样本噪声实现 ε_i，ML代理比较所有可行分区 Π_i 的期望样本外误差（利用验证噪声期望），选择误差最小的分区（若有多个，可随机选择）。
  3. 玩家 i 根据所选分区内的样本均值形成信念，并选择最优纯行动（或混合）。
  4. 将上述步骤对所有噪声实现和玩家整合，得到的混合策略剖面必须与初始 σ 一致，构成不动点。

- **相关均衡概念**：CVE是纳什均衡（无噪声时的极限）和类比预期均衡（ABEE，Jehiel, 2005）的推广。当噪声消失且玩家只能采用最细分区时，CVE收敛到ABEE（若最细分区足够细，则收敛到纳什均衡）。

## 6. 论文的主要结论与发现

- **存在性与收敛性**：在有限博弈中CVE总是存在（Proposition 1）。当训练样本噪声消失时，CVE收敛到关于最细分区的ABEE（Proposition 2）。
- **避免过拟合**：若真实期望充分统计量关于某个分区可测，则ML代理（几乎）不会选择该分区的任意细分（Proposition 3）。对私值独立类型，每个玩家仅使用最粗分区（Corollary 1）。
- **无偏性**：在CVE中，给定所选分区，训练样本噪声的条件期望为零（Proposition 4），因为分区选择基于样本外预测误差且噪声对称。
- **模型选择规则**：当只有最粗和最细两种分区时，选择准则可简化为比较训练样本方差与真实统计量方差（Proposition 5）。
- **战略替代下**（如古诺竞争）：CVE导致行为波动性大于纳什均衡；噪声越大，粗分区使用概率越高，方差越大。存在非对称CVE。
- **战略互补下**（团队努力）：可能出现多重CVE（图1），且即使噪声很小也可能与纳什均衡不连续偏离（δ极小时CVE行为接近恒定均值，远异于纳什）。
- **陪审投票**：CVE可能促使玩家比有效纳什均衡更频繁地投票支持改革，内生模型选择起关键作用。
- **投机赌博**：仅能使用最细分区的玩家可能获得比对手更差的期望收益，说明正确的因果模型不一定占优。
- **控制幻觉**：允许分区包含行动维度后，玩家可能以正概率选择被占优行动，产生“控制幻觉”。
