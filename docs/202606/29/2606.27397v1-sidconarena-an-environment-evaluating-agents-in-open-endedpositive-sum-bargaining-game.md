## 摘要
评估LLM智能体需要超越静态推理和零和博弈的动态环境。现实世界中的经济互动往往是开放式的且动机混合：智能体必须进行谈判、创造正和剩余、竞争稀缺资产，并在延迟回报下进行规划。我们提出了SidConArena，一个新的基准框架，用于在开放、正和谈判中评估LLM智能体。SidConArena将多人经济形式化为一个有限时域部分可观测随机博弈，包含三个耦合阶段：具有约束性交易的自然语言谈判、基于确定性转换器的生产，以及长期资产的密封拍卖。该框架结合了结构化观测、阶段感知智能体调度、神经符号动作接口和异步执行，在保持基于规则评估的同时实现了自由形式的交互。在同类和异类锦标赛中，更强的前沿模型实现了更高的经济结果，但智能体仍然错误估值资源、被动谈判，并在长期投资规划方面受限。

## Abstract
Evaluating LLM agents requires dynamic environments that go beyond static reasoning and zero-sum games. Real-world economic interaction is often open-ended and mixed-motive: agents must negotiate, create positive-sum surplus, compete for scarce assets, and plan under delayed returns. We introduce SidConArena, a new benchmark framework for evaluating LLM agents in open-ended, positive-sum bargaining. SidConArena formalizes a multi-player economy as a finite-horizon partially observable stochastic game with three coupled phases: natural-language negotiation with binding trades, deterministic converter-based production, and sealed-bid auctions for long-term assets. The framework combines structured observations, phase-aware agent dispatching, a neural-symbolic action interface, and asynchronous execution, enabling free-form interaction while preserving rule-grounded evaluation. Across homogeneous and heterogeneous tournaments, stronger frontier models achieve higher economic outcomes, yet agents still misvalue resources, bargain passively, and remain limited in long-horizon investment planning.

- **核心问题**：如何构建一个**动态、部分可观测、多智能体、正和博弈**的评估环境，以检测LLM在资源估值、谈判策略、生产优化和长期投资规划等综合经济能力？

## 2. 方法论

### 核心思想
SidConArena将多智能体经济博弈形式化为**有限时域部分可观测随机博弈**，包含三个耦合阶段：谈判（合作）、生产（确定性优化）、拍卖（竞争非完美信息）。通过**神经符号动作接口**和**阶段感知调度**，在支持自由形式自然语言交互的同时，保证环境执行的规则一致性与可验证性。

### 关键技术细节
1. **问题形式化**：  
   - 博弈元组 \(G = \langle N, S, A, P, H, V_{\text{terminal}}, O, Z \rangle\)  
   - \(N\)：非对称智能体集合；\(H\)：有限时域；\(S\)：全局状态空间；\(A\)：联合动作空间（包含谈判动作 \(A_{\text{neg}}\)、生产动作 \(A_{\text{prod}}\)、投标动作 \(A_{\text{bid}}\)）；\(P\)：转移概率（生产确定性，拍卖随机）；\(V_{\text{terminal}}\)：终端价值函数（线性加权资源向量）；\(O\)：观测空间；\(Z\)：观测函数。

2. **三阶段流程**：
   - **谈判阶段（Phase 1）**：智能体通过自然语言消息 + 绑定交易向量进行资源交换。交易需满足双方启发式效用 \(U\) 的帕累托改进条件。
   - **生产阶段（Phase 2）**：智能体本地选择转换器激活向量，将输入资源转换为输出资源，过程确定性（类似背包问题）。
   - **拍卖阶段（Phase 3）**：密封投标，多轨优先拍卖（Right-of-Choice）。智能体使用“飞船”货币投标殖民地/技术，按出价排序后依次选择并支付溢价。

3. **观测构建**：每个智能体接收四通道观测：私有资源/能力、公开板信息、市场阶段上下文、交互记录（接收和发出的提议）。

4. **智能体架构**：
   - **分层大脑（Brain）**：阶段感知调度器，将不同任务路由到专门调用器（TurnPlanCaller、TradeCaller、EconomyCaller、BidCaller、PickCaller等）。
   - **神经符号动作接口**：智能体仅通过合法函数调用执行动作，环境进行语法验证和状态更新。
   - **异步事件驱动**：采用发布-订阅模式，允许多个智能体并行接收提议并响应，不阻塞游戏循环。

5. **目标函数**：最大化终端时持有的资源总值（线性权重向量 \(w\)），中间时刻使用启发式效用 \(U\) 近似净现值，其中包含未来生产预期收益。

1. **性能分化明显**：更强的前沿模型（GPT-5、Gemini-3）在同质和异质设置中均获得更高终端经济收益；小型或早期模型（如GPT-4o-mini、GLM-4.7）表现显著较差。SidConArena能够区分模型能力。

## 7. 优点

- **创新的评估范式**：首次将开放式正和博弈引入LLM智能体评估，触及真实经济中的混合动机、延迟回报、非对称信息等关键维度。
- **架构设计合理**：
  - 神经符号动作接口兼容自由文本与结构化指令，保证评估可重复。
